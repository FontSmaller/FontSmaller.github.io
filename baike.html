What is font file subsetting 什么是字体文件子集化


Font file subsetting is the operation of extracting some characters from the font file, and then save as a new font file with smaller size.

In web front-end design, special fonts are often required. There are two common methods: one is to use Photoshop or other software to make text layers into pictures. The second is to directly import font files. The first way has obvious disadvantages. Using pictures instead will increase the cost of production and maintenance, and will cause more broadband consumption to the server and users. In addition, users cannot select and copy the text. The second solution solves some of the above problems, but the number of total characters may be too large, such as Chinese, which will result in too large size of font files and slow loading speed.

The appearance of font file subset tool makes it easy for users to extract the required characters from font files, which greatly reduces the size of font files and makes the use of Chinese and other characters fonts more convenient in web front-end design.


字体文件子集化是从字体文件中抽出部分(特定的)字符，再组成新的精减后的字体文件的操作。

在进行前端设计时，经常会需要特殊字体。经常采用的方案有两种：一是使用photoshop等软件将文本图层做成图片；二是直接引入字体文件(.ttf等)。第一种方案缺点明显，使用图片代替文字，制作和维护的成本很高，同时使用图片，更会给服务器端及用户带来更多的宽带消耗。除此之外用户体验方面，用户无法进行选择和复制等操作。第二种方案，虽然解决了上述的一些问题，但是由于一些语言的字体字符数量太大（如中文），会导致字体文件往往过大，字体加载速度很慢，不适合在项目中使用。

字体文件子集化工具的出现，使用户可以便捷的将字体文件中需要的字符提取出来，大大缩小了字体文件的体积，使中文等字体在前端设计中的使用更为便利。
