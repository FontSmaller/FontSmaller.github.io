FontSmaller   字体文件子集化工具   Font File Subset Tool

FontSmaller 字体文件子集化工具是一个免费的程序，它可以帮助用户从一个字体文件中抽出部分字符，再组成精减的字体文件，主要用于网页设计等。

FontSmaller font file subset tool is free. It can help users to extract some characters from a font file (TTF), which is usually used to reduce the size of fonts for website pages.

主要优势：更易操作，无需安装。相较于其他在线版工具，其不受子集字符数量限制。

Features: Easy to use, no installation is required. Compared with other online tools, there is no limitation of the number of subset characters.

Error Code：


Error Code：1x01，Form Load Failed

Detail：System win32-form load failed

Solution：None


Error Code：0x01，Unknown File Format

Detail：This tool is compatible with only TTF (TrueTypeFont) format font files, and other format font files are invalid

Solution：Please confirm that the file you choiced is TTF


Error Code：0x02，Path Error

Detail：System %appdata% directory acquisition failed

Solution：None


Error Code：0x03，Corrupt TrueType Font File

Detail：Unable to read the font file. The main possible cause is file is corrupt

Solution：Try to use another font file instead


Error Code：0x04，Failed to Save File

Detail：Cannot output subset results or preview page

Solution：Run the software with administrator privileges


常见错误代码：


错误代码：1x01，窗体加载失败

原因：系统win32窗体加载失败

解决方案：暂无


错误代码：0x01，未知文件格式

原因：此软件仅支持加载TTF(TrueTypeFont)格式字体文件，暂不支持其他格式字体文件

解决方案：请确认打开加载的文件格式为TTF


错误代码：0x02，路径错误

原因：系统%APPDATA%目录获取失败

解决方案：暂无


错误代码：0x03，TrueType字体文件损坏

原因：无法读取导入的字体文件，主要可能原因为文件损坏

解决方案：尝试更换导入其他字体文件


错误代码：0x04，储存文件失败

原因：无法输出子集化结果或预览网页

解决方案：尝试以管理员身份运行软件
