FontSmaller   字体文件子集化工具   Fonts File Subset Tool

FontSmaller字体文件子集化工具可以让用户抽出一个字体文件中部分想要的字符，再组成精减的字体文件，主要用于网页设计等。

FontSmaller fonts file subset tool help users to extract some characters from a font file (TTF), which is usually used to reduce the size of fonts for website pages.

主要优势：更易操作，无门槛。相较于其他在线版工具，其不受子集字符数量限制。

Features: Easy to operate, no threshold. Compared with other online tools, there is no limitation of the number of subset characters.

常见错误：


错误代码：1x01，窗体加载失败

原因：系统win32窗体加载失败

解决方案：暂无


错误代码：0x01，未知文件格式

原因：此软件仅支持加载TTF(TrueTypeFont)格式字体文件，暂不支持其他格式字体文件

解决方案：请确认打开加载的文件格式为TTF


错误代码：0x02，路径错误

原因：系统%APPDATA%目录获取失败

解决方案：暂无


错误代码：0x03，TrueType字体文件损坏

原因：无法读取导入的字体文件，主要可能原因为文件损坏

解决方案：尝试更换导入其他字体文件


错误代码：0x04，储存文件失败

原因：无法输出子集化结果和预览网页

解决方案：尝试以管理员身份运行软件
